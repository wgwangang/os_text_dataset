# os_text_dataset

A curated catalog of open-source text datasets suitable for training language models.

## Output

`datasets.csv` — columns:

| Column | Description |
|--------|-------------|
| `dataset_name` | Name of the dataset |
| `language` | Language(s) of the data |
| `license` | Open-source license type |
| `url` | Official website or download link |

## Scope

Covers general web crawls, books, Wikipedia, code, dialogue, multilingual, and domain-specific text datasets.
